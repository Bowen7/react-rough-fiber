# Get Started
 
## Introduction

`react-rough-fiber` is a React renderer for rendering hand-drawn SVG.

## Install

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash copy
    pnpm i react-rough-fiber react-reconciler
    ```
  </Tab>
  <Tab>
    ```bash copy
    npm i react-rough-fiber react-reconciler
    ```
  </Tab>
  <Tab>
    ```bash copy
    yarn add react-rough-fiber react-reconciler
    ```
  </Tab>
</Tabs>

<Callout type="warning" emoji="⚠️">
  If you are using React 17, you need to install `react-reconciler@0.26.2`.
</Callout>

## Quick Start

To render the SVG in a hand-drawn style, you only need to add three lines of code:

```jsx {1,4,8}
import { RoughSVG } from 'react-rough-fiber'
export default function App() {
  return (
    <RoughSVG>
      <svg viewBox="0 0 64 64" width="256" height="256">
        <rect x={12} y={12} width={48} height={48} stroke="#000" fill="#82ca9d" />
      </svg>
    </RoughSVG>
  )
}
```

## Integration

### Icons

With [react-feather](https://github.com/feathericons/react-feather):

<RoughSVG className="flex gap-4 my-4" roughOptions={{roughness: 0.6, seed: 2, bowing: 15}}>
  <Battery />
  <Bell />
  <Coffee />
  <Twitter />
  <Camera />
  <Edit />
  <Image />
</RoughSVG>

[Code](/examples/icon)

### Chart

With [recharts](https://github.com/recharts/recharts):

<RoughSVG className="my-4" roughOptions={{fillStyle: 'zigzag'}} style={{fontFamily: 'HandDrawnFont'}}>
  <BarChart width={730} height={250} data={data}>
    <XAxis dataKey="name" />
    <YAxis />
    <Tooltip />
    <Legend />
    <Bar dataKey="pv" fill="#8884d8" stroke="#333" />

</RoughSVG>
